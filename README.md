# Peer Evaluation Google Apps Script App

Workflow changes:

- Students register, then login, then evaluate.
- Passkeys require at least 4 characters.
- Comments require at least 10 characters.
- Faculty creates activity keys before students use them.
- Blank class code means all classes.
- Blank group means all groups.
- Saving an activity opens the rubric editor with the current or default rubric.
